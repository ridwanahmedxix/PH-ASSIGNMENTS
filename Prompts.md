# AI Prompts Used for the Placeholder Sections

## Prompt 1: Conference Schedule Section

Create a clean and modern **"Conference Schedule"** section for a **DevConf 2026** landing page using only **HTML5 and CSS3**.

### Requirements:

- Do not use JavaScript or any CSS framework.
- The design should match a modern technology conference website.
- Include a centered section heading **"Conference Schedule"** with a short subtitle.
- Create three schedule cards displayed in a responsive row.
- Each card should include:
  - Session time
  - Session title
  - A unique short description
- Use a clean white card layout with rounded corners, subtle shadow, blue accent colors, and a smooth hover effect.
- Keep the code semantic, organized, and beginner-friendly.
- Make the section visually consistent with the rest of the DevConf landing page.
